# Web Desktop Sidebar Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add a shadcn/ui desktop sidebar to `apps/web` while preserving the existing mobile bottom navigation.

**Architecture:** Navigation items become shared data used by both mobile and desktop navigation. `AppShell` owns the responsive shell: a fixed sidebar on `md+`, the existing bottom nav below `md`, and a main content area that shifts right only on desktop. The shadcn/ui `sidebar` component is installed through the project registry and used by a focused `DesktopSidebar` component.

**Tech Stack:** React 19, React Router 7, Vite, Tailwind CSS 4, shadcn/ui, lucide-react.

---

## File Structure

- Create: `apps/web/src/components/nav-items.ts`
  - Shared route metadata for mobile bottom nav and desktop sidebar.
- Create: `apps/web/src/components/DesktopSidebar.tsx`
  - Desktop-only sidebar navigation using shadcn/ui sidebar primitives.
- Modify: `apps/web/src/components/MobileBottomNav.tsx`
  - Consume shared nav items instead of declaring duplicate route data.
- Modify: `apps/web/src/components/AppShell.tsx`
  - Render `DesktopSidebar` for desktop and adjust content spacing.
- Create: `apps/web/src/components/ui/sidebar.tsx`
  - Generated by `pnpm dlx shadcn@latest add @shadcn/sidebar`.
- Verify: `apps/web` TypeScript build and Vite production build.

```tsx
import { navItems } from "./nav-items";

export function MobileBottomNav() {
  return (
    <nav className="fixed inset-x-0 bottom-0 z-50 border-t bg-background/94 px-3 pb-[max(env(safe-area-inset-bottom),0.75rem)] pt-2 backdrop-blur md:hidden">

                isActive && "bg-primary text-primary-foreground shadow-sm"
              )
            }
          >
            <item.icon aria-hidden="true" className="h-5 w-5" strokeWidth={2.2} />
            <span>{item.label}</span>
          </NavLink>
        ))}
      </div>
    </nav>
  );
}
```

### Task 3: Add Desktop Sidebar Component

```tsx
import { Radio } from "lucide-react";
import { NavLink } from "react-router";

import {
  Sidebar,
  SidebarContent,
  SidebarFooter,
  SidebarGroup,
  SidebarGroupContent,
  SidebarGroupLabel,
  SidebarHeader,
  SidebarMenu,
  SidebarMenuButton,
  SidebarMenuItem

import { cn } from "@/lib/utils";

import { navItems } from "./nav-items";

  return (
    <Sidebar className="hidden border-r bg-sidebar text-sidebar-foreground md:flex" collapsible="none">
      <SidebarHeader className="border-b px-4 py-4">
        <div className="flex items-center gap-3">
          <div className="flex size-9 items-center justify-center rounded-md bg-sidebar-primary text-sidebar-primary-foreground">
            <Radio aria-hidden="true" className="size-5" strokeWidth={2.2} />
          </div>
          <div className="min-w-0">
            <p className="truncate text-sm font-semibold leading-5">Holyrics Control</p>
            <p className="truncate text-xs text-sidebar-foreground/65">Controle local</p>
          </div>
        </div>
      </SidebarHeader>

      <SidebarContent className="px-2 py-4">
        <SidebarGroup>
          <SidebarGroupLabel>Navegacao</SidebarGroupLabel>
          <SidebarGroupContent>
            <SidebarMenu>
              {navItems.map((item) => (
                <SidebarMenuItem key={item.to}>
                  <SidebarMenuButton asChild tooltip={item.label}>
                    <NavLink
                      to={item.to}
                      end={item.to === "/"}
                      className={({ isActive }) =>
                        cn(
                          "text-sidebar-foreground/75",
                          isActive && "bg-sidebar-primary text-sidebar-primary-foreground"
                        )
                      }
                    >
                      <item.icon aria-hidden="true" className="size-4" strokeWidth={2.2} />
                      <span>{item.label}</span>
                    </NavLink>
                  </SidebarMenuButton>
                </SidebarMenuItem>
              ))}

          </SidebarGroupContent>
        </SidebarGroup>
      </SidebarContent>

      <SidebarFooter className="border-t px-4 py-3">
        <p className="text-xs leading-5 text-sidebar-foreground/60">Desktop</p>
      </SidebarFooter>
    </Sidebar>
  );
```
